# 게시판 서비스 v1.0

기본적인 기능을 제공하는 간단한 게시판 기능을 제공하는 샘플 애플리케이션

## 개발 환경

- Intellij IDEA Ultimate 2022.1.1 ~ 2022.1.3
- Java 17
- Gradle 7.4.1
- Spring Boot 2.7.0

## 기술 세부 스택

- Spring Boot
    - Spring Boot Actuator
    - Spring Web
    - Spring Data JPA
    - Rest Repositories
    - Rest Repositories HAL Explorer
    - Thymeleaf
    - Spring Security
    - H2 Database
    - Lombok
    - Spring Boot DevTools
    - Spring Configuration Processor
- QueryDSL 5.0.0
- Bootstrap 5.2.0
